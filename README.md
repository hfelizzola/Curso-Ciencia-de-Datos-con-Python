# 📊 Ciencia de Datos con Python

Repositorio del curso **Ciencia de Datos con Python** (Ingeniería Industrial – Electiva Disciplinar I).  
Contiene notebooks, datasets y proyectos orientados al **análisis, modelado y despliegue de productos de datos** aplicados a contextos industriales y organizacionales.

---

## 🎯 Objetivo

Desarrollar capacidades para **analizar datos, entrenar modelos de machine learning y construir productos de datos reproducibles** usando Python.

---

## 🧠 Metodología

Curso práctico basado en **aprendizaje basado en proyectos (ABP)**:
- Talleres en notebooks
- Dos proyectos parciales
- Un **proyecto final desplegable** (Streamlit)

---

## 🗓️ Contenidos principales

- Fundamentos de Python para ciencia de datos  
- NumPy y Pandas  
- Limpieza y visualización de datos  
- Análisis exploratorio de datos (EDA)  
- Modelos predictivos (regresión y clasificación)  
- Aprendizaje no supervisado  
- Minería de texto  
- Productos de datos con **Streamlit**  

---

## 🛠️ Tecnologías

Python · Pandas · NumPy · Matplotlib · Seaborn · Scikit-Learn · Streamlit · Jupyter / Colab

## 📦 Resultado final

Un **producto de datos funcional** que integra análisis, modelado y visualización, aplicado a un problema real.
